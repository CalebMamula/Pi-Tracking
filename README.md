# Pi-Tracking
A Unity application capable of tracking digits of Pi by clicking Space or Left Mouse Button. Made with the intention of being used to keep track of how far someone gets during a Pi Recitation Contest. 

The program has the first 10,000 digits of pi from https://calculat.io/en/number/download-pi-digits-files-txt-zip. 

It can all be reset by pressing R.

*There are known bugs, such as the highlighter not showing properly on digit 201. 401, etc. There could be others as this was just a quick fun thing thrown together in a couple hours.*
